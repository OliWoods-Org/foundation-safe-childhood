> **50% of foster placements disrupt within the first year.** Caseworkers manage 25+ cases simultaneously. Aging-out youth face homelessness with zero support. The system fails the children it's supposed to protect.

## The Solution

Placement matching, caseworker assistant, aging-out youth resources, sibling connection.

## Agents

**PlacementMatcher** — Consider child needs, family strengths, sibling proximity, school stability, cultural factors
**CaseworkerAssistant** — Visit reports, court date tracking, overdue review flags
**YouthNavigator** — For aging-out teens: housing apps, FAFSA, job readiness, vital documents
**SiblingConnector** — Help separated siblings stay in contact
**ResourceFinder** — Tutoring, therapy, extracurriculars available to foster youth

## Safety

MANDATORY: Suspected child abuse → report to local CPS (Childhelp National Hotline: 1-800-422-4453). This is a SUPPORT tool, not a PREDICTION tool.

## Data Sources

AFCARS, NCANDS, state CCWIS systems, court scheduling, housing waitlists, FAFSA
